Симулятор ретро-комп'ютера УКНЦ та Pascal/Fast на основі Free Pascal.

Писати програми одразу на ретро-комп'ютерах не завжди буває зручно і швидко,
тому іноді намагаються писати на більш сучасних комп'ютерах, використовуючи різні підходи.

Цю невеличку річ написано для того, щоб використовуючи Free Pascal можна було написати нескладну програму
для УКНЦ Pascal/Fast v2.2-3 з використанням CGRAPH.PAS

Ідея написати це з'явилась після однієї теми на форумі "української спільноти компʼютерної історії",
Список форумів :: Наш полігон :: Програмування під ретроплатформи :: Чи є хтось, хто б міг допомогти по УКНЦ
від 23 травня 2025 р.

Спочатку мені здалося, що це дуже просто і займе зовсім небагато часу, втім, я помилився,
але, все ж, написав дещо.

Синтаксис Free Pascal та УКНЦ Pascal/Fast іноді може відрізнятися, тому періодично потрібно перевіряти
результат компіляції на емуляторі УКНЦ, в даному випадку, https://github.com/nzeemin/ukncbtl/ від Nikita Zimin.

PAS1.SAV, версія Pascal/Fast 2-2.3 взято з УКНЦ архіву hobot з сайту pdp-11 ru (кнопка на сторінці сайту зверху "Раздел УК-НЦ и ДВК") ukncbtlwebwebcomplekt/FASTPASCAL_v221_v223 - UKFP_v221_v223.DSK

Але, UKFP_v221_v223.DSK не бажано використовувати як завантажувальний, бо буде лише монохромний екран, не знаю як це красиво виправити.
Як варіант, скопіювати PAS1 та CGRAPH на якийсь простий завантажувальний диск, наприклад на rt1150sg.dsk, для того щоб були кольори
і було більше місця для компіляції.
